# egg-vue-ssr

基于 Egg + Vue + Webpack SSR 服务端渲染工程骨架项目，包括前台系统(SSR MPA)和后台管理系统(SSR SPA)。 如果要深入了解，建议请先阅读 https://www.yuque.com/easy-team/egg-vue 专题。

### egg入门文档

https://www.kancloud.cn/vipbic/egg-js/1018598


### 安装依赖

npm i

### 开发模式

npm run dev

### 打包

npm run build

### 路由

查看app/router.js
